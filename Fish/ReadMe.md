**Author:** Hazel
**Version:** 0.30 
**Description:**  
釣りを繰り返す


**Abbreviation:** //fish

**Commands:**
 1. fish start		-- 以後、釣りを繰り返す
 2. fish stop		-- 停止する
 3. fish autostop	-- 疲れがたまったら止める(未実装)
 4. fish release add/remove fishName zoneId-- リリースする対象を追加/削除(未実装)
 5. fish reset		-- 疲れカウントクリア(未実装)
 6. fish r			--	設定ファイルの再読み込み(Release対象を追加後など)

Ver 0.3
	・一定回数何も釣れなかったら、動作停止
	・エリア毎に釣った獲物を記録
	・外道はリリース対象として自動記録(１度は釣り上げます)
	・エリア毎にリリースする獲物を設定可能
