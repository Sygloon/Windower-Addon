**Author:** Hazel
**Version:** 0.40 
**Description:**  
釣りを繰り返す


**Abbreviation:** //fish

**Commands:**
 1. fish start		-- 以後、釣りを繰り返す
 2. fish stop		-- 停止する
 3. fish autostop	-- 疲れがたまったら止める(未実装 -> 廃止予定)
 4. fish release add/remove fishName zoneId-- リリースする対象を追加/削除(未実装/手編集で良くない？)
 5. fish reset		-- 疲れカウントクリア(未実装)
 6. fish r			-- 設定ファイルの再読み込み(Release対象を編集後などに)

Ver 0.4
	・釣り時に、登録しておいた装備に着替える
	・エリア時にエリア名称を表示(海賊船も明示)
	・釣った引数を表示＆日またぎでリセット

Ver 0.3
	・一定回数何も釣れなかったら、動作停止
	・エリア毎に釣った獲物を記録
	・エリア毎にリリースする獲物を設定可能
